# Logwise Components

This section provides detailed documentation for each component of the Logwise system.

## Components

- **[Vector](./vector)** - Log collection and forwarding component
- **[Kafka](./kafka)** - High-throughput message broker for log streaming
- **[S3 + Athena](./s3-athena)** - Cost-efficient log storage and querying
- **[Apache Spark](./spark)** - Stream processing and transformation engine
- **[Grafana](./grafana)** - Visualization and dashboards
- **[Orchestrator Service](./orchestrator)** - Central control component for system orchestration

For a high-level overview of how these components work together, see the [Architecture Overview](../architecture-overview.md).
